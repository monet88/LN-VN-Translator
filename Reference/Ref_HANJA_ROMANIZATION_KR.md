## Overview

Korean names in Murim novels use Hanja (漢字) with Korean pronunciation. The **Revised Romanization of Korean (국어의 로마자 표기법)**, adopted in 2000, is the official standard.

**Key Principles:**
1. No diacritics (unlike McCune-Reischauer)
2. Vowels written as-is (no breves/macrons)
3. Consonant assimilation reflected in spelling
4. Names may use traditional spellings (flexibility)

## Sound Change Rules (음운 변화)

### Rule 1: Consonant Assimilation (자음동화)

When consonants meet, they may change sound. RR2000 reflects pronunciation.

| Pattern | Romanization | Example |
|---------|--------------|---------|
| ㄱ + ㄴ | ngn | 학년 = hangnyeon |
| ㄱ + ㄹ | ngn | 독립 = dongnip |
| ㄱ + ㅁ | ngm | 국민 = gungmin |
| ㄴ + ㄹ | ll | 신라 = Silla |
| ㄹ + ㄴ | ll | 설날 = Seollal |
| ㅂ + ㄴ | mn | 십년 = simnyeon |
| ㅂ + ㄹ | mn | 협력 = hyeomnyeok |
| ㅂ + ㅁ | mm | 입문 = immun |

**Name Examples:**
```
남궁 = Namgung (not Namkung)
황보 = Hwangbo (not Hwangpo)
```

### Rule 2: Aspiration (격음화)

ㄱ/ㄷ/ㅂ/ㅈ + ㅎ = ㅋ/ㅌ/ㅍ/ㅊ

| Pattern | Result | Example |
|---------|--------|---------|
| ㄱ + ㅎ | kh → k | 북한 = Bukhan |
| ㄷ + ㅎ | th → t | - |
| ㅂ + ㅎ | ph → p | - |
| ㅈ + ㅎ | ch | - |

### Rule 3: Palatalization (구개음화)

ㄷ/ㅌ + 이 = 지/치

| Pattern | Result | Example |
|---------|--------|---------|
| ㄷ + 이 | ji | 굳이 = guji |
| ㅌ + 이 | chi | 같이 = gachi |

### Rule 4: ㄹ Linking

ㄹ between vowels = r (not l)

| Position | Roman | Example |
|----------|-------|---------|
| Initial | r | 류 = Ryu |
| Medial (V_V) | r | 나라 = nara |
| Final | l | 달 = dal |
| Before consonant | l | 발전 = baljeon |

## Name Consistency Rules

### Rule 1: One Name, One Spelling

Once a character's name is romanized, use it consistently throughout.

**Wrong:**
```
Cheong-myeong → Chungmyeong → Chung Myung
```

**Correct:**
```
Cheongmyeong (throughout entire translation)
```

### Rule 2: Hyphenation

RR2000 allows flexibility. For Murim translation, use:

| Style | Example | Use Case |
|-------|---------|----------|
| No hyphen | Cheongmyeong | Default for given names |
| Hyphen | Cheong-myeong | Compound clarity |
| Space | Cheong Myeong | Western-style format |

**Recommendation:** No hyphen for cleaner reading.

### Rule 3: Surname-Given Name Order

Maintain Korean order (surname first) in Murim context.

**Example:**
```
정청명 = Jeong Cheongmyeong (not Cheongmyeong Jeong)
```

### Rule 4: Traditional Spellings

Some surnames have established English spellings that differ from RR2000.

| RR2000 | Traditional | Usage |
|--------|-------------|-------|
| Bak | Park | Common in real names |
| I | Lee | Common in real names |
| Choe | Choi | Common in real names |
| Sin | Shin | Common in real names |
| Im | Lim | Common in real names |

**For Murim:** Prefer RR2000 for consistency, but note traditional in glossary.

### Rule 5: Title + Name Format

| Korean | Format | Example |
|--------|--------|---------|
| 장문인 | Title Name | Jangmunin Cheongmyeong |
| 대협 | Title (Name) | Daehyeop (Cheongmyeong) |
| 청명 공자 | Name Title | Cheongmyeong Gongja |

## Tricky Romanizations

### ㄹ Initial vs Medial

| Position | Roman | Example |
|----------|-------|---------|
| Initial | R | 류 = Ryu, 이 = I |
| Medial | R | 나라 = nara |
| Final | L | 철 = Cheol |

**Exception:** 이 (李) → I or Lee (traditional)

### Double Consonants

| Hangul | Roman | Example |
|--------|-------|---------|
| ㄲ | kk | 꽃 = kkot |
| ㄸ | tt | 딸 = ttal |
| ㅃ | pp | 빵 = ppang |
| ㅆ | ss | 쌀 = ssal |
| ㅉ | jj | 짜다 = jjada |

### ㅇ Silent vs NG

| Position | Sound | Roman | Example |
|----------|-------|-------|---------|
| Initial | Silent | - | 아 = a |
| Final | ng | ng | 강 = gang |

### ㅢ Variations

| Context | Pronunciation | Roman |
|---------|---------------|-------|
| Word-initial | ui | 의사 = uisa |
| After consonant | i | 희망 = himang |
| Possessive marker | e | 나의 = nae |
