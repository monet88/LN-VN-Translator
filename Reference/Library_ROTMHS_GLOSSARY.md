## Category 1: Gangho Core Terminology

### Fundamental Concepts

**강호 (江湖) - Gangho / Martial World**
- Hán Việt: Giang hồ
- Definition: Society of martial artists, separate from civilian world
- Scope: All sects, clans, warriors operating under martial law
- Counterpart: 양민 (良民 - common people's world)
- Example: "강호는 힘이 지배하는 세계다" → "Gangho is a world ruled by strength"

**정파 vs 사파 (Righteous vs Evil)**

| Term | Hanja | English | Hán Việt | Description |
|------|-------|---------|----------|-------------|
| 정파 | 正派 | Jeongpa | Chính phái | Righteous factions (orthodox sects) |
| 사파 | 邪派 | Sapa | Tà phái | Evil factions (unorthodox sects) |
| 사도 | 邪道 | Sado | Tà đạo | Evil Path methodology |
| 사기 | 邪氣 | Sagi | Tà khí | Energy associated with evil sects |

**Demonic Faction:**
- 마교 (魔敎) - Magyo / Demonic Cult (Ma giáo)
- 마도 (魔道) - Mado / Demonic Path (Ma đạo)
- 마기 (魔氣) - Magi / Demonic energy (Ma khí)
- 마강 (魔剛) - Magang / Superior attack demonic force (Ma cương)

### Martial Arts Fundamentals

**무학 (武學) - Muhak / Martial Arts**
- Hán Việt: Võ học
- Scope: Systematic study of combat arts
- Includes: 검법 (sword), 장법 (palm), 권법 (fist), 경공 (qinggong)

## Usage Notes

### Translation Patterns

**Character Names:**
- Maintain Korean romanization (Chung Myung, not Cheong-myeong)
- Use generational naming (Hyun-, Un-, Baek-)

**Technique Names:**
- Keep Korean Hanja terms for signature techniques
- Provide Hán Việt in parentheses on first mention
- Example: "매화검법 (Mai Hoa Kiếm Pháp)"

**Honorifics:**
- Context-dependent: 사형 (Sahyeong) for internal sect, 도장 (Dojang) for external
- Maintain Korean terms for sect-specific address

### Common Patterns

**Address Formulas:**
- Within sect: [Name] + 사형/사제/사숙 (generational term)
- External polite: [Name/Title] + 도장/대협/가주 (respectful title)
- Casual: [Name] alone or [Name] + 형 (hyeong - brother)

**Technique Descriptions:**
- Korean term + brief description + Hán Việt
- Example: "자하신공 (Tử Hà Thần Công) - Mount Hua's supreme internal art"

**Combat Intensity Scaling:**
- 검기 (Sword qi) → 검강 (Sword energy) → 검력 (Sword force)
- Indicate power progression in descriptions
