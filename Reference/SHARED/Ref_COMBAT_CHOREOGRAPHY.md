## Overview

Korean Murim combat choreography differs from other wuxia traditions. Key characteristics:

1. **Staccato Rhythm**: Short, punchy sentences dominate action scenes
2. **Energy-Physical Integration**: Qi attacks flow naturally with physical moves
3. **Realm-Scaled Impact**: Higher cultivation = more cosmic language
4. **Economical Description**: Show decisive moments, skip filler

This reference provides 15+ reusable patterns for translating extended fight scenes.

## Pattern Types (15+ Patterns)

### Pattern 1: Opening Strike

**Context:** First move of a fight, sets the tone.

**Structure:**
```
[Tension] → [First move] → [Reaction]
```

**Korean Example:**
```korean
정적이 흘렀다.
청명이 먼저 움직였다.
검이 번쩍였다.
```

**Vietnamese Translation:**
```vietnamese
Tĩnh lặng bao trùm.
Chung Myung ra tay trước.
Kiếm lóe sáng.
```

**Usage Notes:**
- Keep opening simple, build complexity later
- Tension phrase is optional but adds weight
- First mover often dictates fight rhythm

### Pattern 2: Sword Exchange

**Context:** Back-and-forth clash between two swordsmen.

**Structure:**
```
[Strike]. [Block]. [Counter]. [Dodge]. [Strike]. [Result].
```

**Korean Example:**
```korean
검이 부딪쳤다. 불꽃이 튀었다.
막았다. 비틀었다. 찔렀다.
상대가 피했다. 다시 공격했다.
```

**Vietnamese Translation:**
```vietnamese
Kiếm va kiếm. Lửa bắn.
Chặn. Xoay. Đâm.
Đối phương né. Công tiếp.
```

**Usage Notes:**
- Use fragment sentences, avoid connectors
- Each action = one sentence or phrase
- Sparks/fire = standard sword clash imagery

### Pattern 3: Power Gap Demonstration

**Context:** Strong fighter overwhelms weaker opponent.

**Structure:**

**Vietnamese Translation:**
```vietnamese

Chung Myung nhẹ nhàng né. Dùng ngón tay gạt kiếm.
```

**Usage Notes:**
- Contrast effort levels (필사적 vs 가볍게)
- Dismissive action/dialogue reinforces gap
- Humiliation verbs: 쳐냈다, 흘려보냈다, 튕겨냈다

## Energy Integration Guide

### Principle: Qi and Physical Actions Interweave

In Korean Murim, internal energy (내공) is not separate from physical combat. Every strike can carry qi, and qi attacks often accompany physical movement.

### Integration Methods

#### Method 1: Pre-Attack Qi Gathering

```korean
내공을 검에 실었다. 찔렀다.
```
```vietnamese
Dồn nội lực vào kiếm. Đâm.
```

**Rule:** Gathering happens in previous sentence, attack in next.

#### Method 2: Simultaneous Qi-Physical

```korean
검기와 함께 검을 내리쳤다.
```
```vietnamese
Chém xuống kèm kiếm khí.
```

**Rule:** Use "와 함께" or "kèm" for simultaneity.

#### Method 3: Qi as Standalone Attack

```korean
검을 휘두르자 검강이 날아갔다.
```
```vietnamese
Vung kiếm. Kiếm cương phóng ra.
```

**Rule:** Physical motion triggers qi projection.

#### Method 4: Qi Defense Layer

```korean
호신강기를 펼쳤다. 검이 튕겨나갔다.
```
```vietnamese
Triển khai hộ thân cương khí. Kiếm bật ra.
```

**Rule:** Defensive qi mentioned, then effect shown.

## Pacing Guidelines

### Combat Rhythm by Scene Type

#### Quick Skirmish (< 5 exchanges)

```
Fragment. Fragment. Fragment. Done.
```

- Maximum speed
- Minimal description
- 1-2 sentences per action
- Total: 50-100 words

**Example:**
```vietnamese
Vung kiếm. Chém. Đối phương đỡ. Xoay. Đâm. Gục.
```

#### Standard Duel (10-20 exchanges)

```
[Opening]. [Exchange block]. [Turning point]. [Climax]. [Resolution].
```

- Varied sentence length
- Some technique names
- Environmental hints
- Total: 200-400 words

**Example Structure:**
1. Opening stance (20 words)
2. First clash (40 words)
3. Back-and-forth (80 words)
4. One side gains advantage (40 words)
5. Desperate counter (40 words)
6. Final exchange (30 words)
7. Result (10 words)

#### Extended Battle (30+ exchanges)

```
[Phase 1: Probing]. [Phase 2: Escalation]. [Phase 3: Crisis]. [Phase 4: Resolution].
```

- Multiple rhythm changes
- Breather moments
- Technique showcase
- Total: 500-1000 words

**Phase Guidelines:**
- Probing: Short sentences, testing
- Escalation: Longer descriptions, techniques appear
- Crisis: Injury, desperation, staccato returns
- Resolution: Decisive exchange, brief

### Sentence Length Variation

| Moment Type | Sentence Length |
|-------------|-----------------|
| Standard action | 3-8 words |
| Technique activation | 10-15 words |
| Energy description | 8-12 words |
| Injury/pain | 5-10 words |
| Dialogue in combat | 3-7 words |
| Climax moment | 2-5 words |

### Punctuation Rhythm

| Punctuation | Use Case |
|-------------|----------|
| Period (.) | Standard beat, most common |
| Exclamation (!) | Impact, pain, technique call |
| Ellipsis (...) | Pause before reveal |
| Em-dash (—) | Interrupted action |
